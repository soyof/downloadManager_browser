## [1.0.6] - 2025-12-25

### 功能优化

- 🔧 优化文件详情弹窗，移除下载ID字段，添加下载状态、进度、速度、文件存在、可恢复下载等更多有用信息
- 🔧 优化错误信息显示，将 Chrome 错误代码（如 USER_CANCELED）翻译为用户友好的中文提示

- 🌐 添加错误信息相关翻译键（downloadError、downloadCanResume、downloadReceivedBytes）到所有语言文件
- 🌐 完善错误代码翻译，支持所有常见 Chrome 下载错误代码的中文显示

### 代码优化

- 🧹 优化日期格式化函数，使用手动格式化确保日期格式统一
- 🧹 优化错误信息翻译函数，统一错误代码转换逻辑

## [1.0.5] - 2025-12-23

### 新增功能

- ✨ 添加文件详情弹窗功能，显示文件的完整详细信息
- ✨ 添加文件详情按钮，仅在已完成且存在的文件显示

- 🔧 优化文件类型显示格式，显示为"文档（xlsx）"形式，更直观
- 🔧 优化时间显示格式，开始时间和结束时间精确到秒
- 🔧 优化文件详情弹窗样式，增加宽度、添加滚动条、优化布局
- 🔧 优化文件详情按钮图标，使用 View 图标，与按钮组风格一致
- 🔧 优化弹窗打开/关闭动画，添加弹跳动画效果，提升用户体验

### 代码优化

- 🧹 扩展 DownloadItem 类型定义，添加 referrer、incognito、danger 字段
- 🧹 优化下载项转换器，保留更多 Chrome API 字段信息
- 🧹 封装文件详情弹窗为独立组件（FileDetailsDialog），提升代码可维护性
